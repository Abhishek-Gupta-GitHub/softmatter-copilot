# Soft Matter Physicist's Copilot

Upload confocal data → detect particles (DeepTrack2) → track (trackpy) → compute MSD/RDF → get a physics summary.

## Quick start
- Create a conda env
- Install requirements
- Run: `python scripts/run_demo.py`

## Folder structure
- src/      core Python code
- notebooks/  exploration and demos
- examples/   small test datasets
